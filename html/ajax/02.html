02 AJAX使用

一、AJAX开发步骤

  1. 在浏览器内存中，创建一个脚本对象，这个脚本对象就是异步请求对象
  2. 为异步请求对象添加工作状态监听器帮助开发人人员何时从异步请求对象身上得到服务端返回响应数据
  3. 初始化异步请求对象
    1. 通知异步请求对象采用何种方式请求方式发送请求协议包（POST/GET）
    2. 通知异步请求对象本次要访问的资源文件地址
    3. 通知异步请求对象在其工作期间，浏览器是否会等它。（同步请求，异步请求）
  4. 命令通知异步请求对象代替浏览器发送请求

二、同步请求和异步请求区别

  1. 同步请求

    在当前异步请求对象工作期间，浏览器只能处于等待状态，直到当前异步请求对象工作完毕，浏览器才可以继续委派其它异步请求对象发送请求

  2. 异步请求

    在当前异步请求对象工作期间，浏览器可以继续委派其它异步请求对象发送其他请求

三、AJAX命令

  1. 如何创建一个异步请求对象
    var xmlHttp = new XMLHttpRequest();
  2. 如何为异步请求对象绑定工作状态监听器 obj.监听器 = function()
    xmlHttp.onreadystatechange = function(){·····};
  3. 如何初始化异步请求对象
xmlHttp.open("post/get","url","true/false")
/**
*第一个参数代表采用的请求方式
*第二个参数代表要访问的地址
*第三个参数表示采用同步还是异步请求方式，true：异步     false：同步
*/
  1. 如何命令异步请求对象代替浏览器发送请求
    xmlHttp.send();

四、异步请求对象5种工作状态

  1. readyState属性：存储当前工作状态
  2. 含义：
    xmlHttp.readyState 状态 发生位置
    0 异步请求对象已经创建完毕 var xmlHttp = new XMLHttpRequest()
    1 异步请求对象已经被初始化完毕 xmlHttp.open("post/get","url","true/false")
    2 异步请求对象已经发送了请求协议包。如果正常情况此时服务端应该正在处理这个请求 xmlHttp.send()
    3 异步请求对象已经得到服务端返回的响应包，此时异步请求对象正在解析响应包中响应体内容 ···············
    4 异步请求对象已经解析数据完毕，此时数据符合脚本规范的数据。 开发人员可以提取响应数据

五、如何从异步请求对象身上得到响应数据，进行局部刷新

xmlHttp.onreadystatechange = function(){
    //每当事件处理函数被调用时，都需要询问异步请求对象当前工作状态
    if(xmlHttp.readyState==4 && xmlHttp.status==200){
        //读取响应数据
        var data = xmlHttp.responseText;
        //调用回调函数处理返回结果
        callBack(data)
    }
}

//局部刷新回调函数
function callBack(param){
    $("div").text(param)
}

六、如何将服务端高级类型数据交给异步请求处理函数

  1. 异步请求对象和浏览器无法解析服务端返回的高级类型对象
  2. 异步请求对象和浏览器只能接受来自于服务端发送的基本数据类型（int，String）
  3. 只能将高级类型对象转换为标准JSON格式字符串交给异步请求对象处理
