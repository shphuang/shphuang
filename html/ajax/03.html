03 在jQuery中使用AJAX

一、jQuery对AJAX开发步骤的封装版

体现思路

  • 不需要开发人员亲自实现AJAX的四步骤
  • 只需要开发人员提供AJAX开发步骤需要的参数
  • 可以让开发人员专注局部刷新实现

二、$.ajax()：是jQuery为AJAX开发提供的标准简化版

  1. 使用:

ajax()只接受一个json对象作为参数。开发人员可以将AJAX实现需要的参数在json对象中进行声明

  1. 参数声明
var option = {
    type:
        [作用]：通知异步请求对象采用何种方式发送请求协议包（post/get）
        [赋值]：type:'get'  ,  type:'post'
        [注意]：type属性可以不声明，默认异步请求对象使用GET请求方式
    
    url:
        [作用]：本次访问的资源文件地址，要求异步请求对象将该地址写入请求协议包
        [赋值]：url:'/web/····'
        [注意]：url属性不能省略，url属性不能携带参数

    data:
        [作用]：通知异步请求对象本次请求携带的参数
        [赋值]：
                1. data属性可以接受字符串
                    data:"uname=john&password=123"
                2. data属性也可以接受json格式描述参数
                    data:{
                        "uname":"john",
                        "password":"123"
                    }
        [注意]：如果没有要携带的参数，可以不声明

    asyn:
        [作用]：设置同步请求与异步请求
        [赋值]：
                async:true , 异步请求
                async:false , 同步请求
        [注意]：该参数可以不声明，默认使用异步请求

    success:
            [作用]：指定异步请求对象工作完毕后，需要调用的【局部刷新函数名】
            [赋值]：success:callback
            [注意]：只写局部刷新函数名，不需要带()，该参数不能省略

    error:
        [作用]：通知异步请求对象在接受到【404、405、500】等状态码，需要调用的方法名
        [注意]：开发测试会写，上线不需要

    dataType:
            [作用]：通知jQuery将异步请求对象中读取数据转变为何种数据
            [赋值]：
                    dataType:'text',文本类型
                    dataType:'json',json类型
            [注意]： 该参数可以不声明，默认是文本类型  

}
  1. 演示
//表单
<form method="get">
    <input type="text" name="uname" id="uname" onblur="fun1()">
</form>

//脚本
<script>
    //回调函数
    function callback(data) {
        alert(data)
    }


    function fun1(){
        var val = $("#uname").val();

        //json对象
        var option = {
            type:'get',//请求方式GET
            url:'/myWeb/one.do',//请求地址
            data:'uname='+val,//请求参数
            success:callback,//回调函数
            async:true,//异步请求
            dataType:'text'//以文本类型接收
        }

        //使用AJAX
        $.ajax(option)
    }

</script>

三、$.get()与$.post()

  1. 介绍

    $.get():通知异步请求对象采用GET方式发送请求
    $.post():通知异步请求对象采用POST方式发送请求

  2. $.get()使用

  • 基本格式：$.get(请求地址,请求参数,局部刷新函数,需要的数据类型【text/json】)
  • 如果只得到文本类型格式：：$.get(请求地址,请求参数,局部刷新函数)
  • 如果不携带参数和只需要得到文本数据格式：：$.get(请求地址,局部刷新函数)
