04 git本地库常用操作（二）

一、git分支的创建与删除

  1. 创建新的分支
    使用命令：git branch [分支名]
  2. 查看分支的个数
    使用命令：git branch -v
  3. 切换分支
    使用命令：git checkout [分支名]
  4. 删除分支
    使用命令：git branch -d [要删除的分支名]

二、git分支的合并与冲突解决

  1. 合并分支
  • 切换到接受修改的分支
    使用命令：git checkout [被合并分支名]
  • 执行merge命令
    使用命令：git merge [有新内容的分支名]
  1. 解决合并分支冲突
    冲突表现：合并出现错误
    冲突解决：
  • 编辑文件，删除特殊符号
  • 把文件修改到满意的程度，保存退出
  • git add [文件名]
  • git commit -m "注释"
    注意：此时commit不能带具体文件名

三、ssh免密登录

  1. 设置系统级别签名

git config --global user.name [用户名]
git config --global user.email [用户邮箱（建用使用注册码云的邮箱）]

  1. 进入git用户在该系统的home目录:cd ~
  2. 查看是否有.ssh目录，使用命令：ls -a
  3. 如果没有则初始化.ssh,使用命令：ssh-keygen -t rsa -C [邮箱名]。后面会有几个选项，按enter默认即可
  4. 初始化ssh后进入.ssh/目录并查看该目录的文件
  5. 查看id_rsa.pub文件并复制该文件的内容，查看命令：cat id_rsa.pub
  6. 登录gitee,进入设置并找到ssh公钥选项进入。
  7. 填写公钥并保存
  8. 尝试与gitee通信，输入命令：ssh -T git@gitee.com，待会儿会询问是不是要把远端地址加入到konw_host中，如果直接回车，则默认没有权限写入，必须输入yes才能正确写入 konw_host。
  9. 进入你要push的仓库，复制你的ssh地址
  10. 使用push向gitee推送文件。
    使用命令：git push [gitee仓库的ssh地址] [分支]
